# dirbrute
Simple Python3 Script for Brute Force Attack To Web Directories

Requirements:
requests
#pip3 install requests

Usage:
./Dirbrute.py <target> <wordlist_file> <threads>
 ex: ./Dirbrute.py http://127.0.0.1 /usr/share/wordlists/dirb/common.txt 10
